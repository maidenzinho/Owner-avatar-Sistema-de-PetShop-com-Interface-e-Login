# Sistema de PetShop com Interface e Login
### Bibliotecas necessárias:
```
pip install sqlalchemy
```
```
pip install pyqt5
```
```
pip install pymysql
```
```
pip install hashlib
```

### O sistema possui login de funcionário e de admin:

### Admin
```
Usuário: admin
Senha: senha123
```

### Funcionário
```
Usuário: funcionario
Senha: senha456
```
